📡 SMS Kurulumu
ADIM 1 / 3

MacroDroid yüklüyse açılır, yüklü değilse Play Store'a yönlendirilir.

ADIM 2 / 3

MacroDroid'de SMS → GAS makrosunu kurun:

1️⃣ MacroDroid → + → Makro Ekle
2️⃣ Tetikleyici: SMS/MMS Alındı
İçerik içerir: sim.csb.gov.tr
3️⃣ Eylem: Bağlantı → HTTP İsteği
• URL: 📋 GAS URL (kopyalanan)
• Yöntem: POST
• İçerik Türü: application/json
• Gövde: 📋 JSON Gövde (kopyalanan)
4️⃣ Makroya isim ver → Kaydet → Etkinleştir
ADIM 3 / 3

Backend bağlantısını test edin.

⚡ Otomatik İletim — iOS Kısayollar

Telefona SİM SMS'i gelince otomatik olarak GAS'a iletilir. Kısayollar uygulaması iOS'ta zaten yüklü gelir.

1️⃣ Kısayollar uygulamasını aç
2️⃣ Otomasyon sekmesi → + → Kişisel Otomasyon
3️⃣ Mesaj tetikleyicisini seç
İçerik içerir: sim.csb.gov.tr
4️⃣ Eylem Ekle → URL İçeriğini Al
• URL: 📋 GAS URL (aşağıdan kopyala)
• Yöntem: POST • Gövde Tipi: JSON
• Alan1: action = smsWebhook
• Alan2: body = [Kısayol Girdisi]
• Alan3: tesis = ozdere
5️⃣ Çalıştırmadan Önce Sor → KAPAT
6️⃣ Bitti ✅
🔔 Ücretsiz Anlık Bildirim — ntfy.sh

2FA kodu geldiğinde anında bildirim alın. Ücretsiz, kurulumu kolaydır.

1️⃣ Aşağıdan ntfy uygulamasını App Store'dan indirin
2️⃣ Uygulamayı açın → sağ üstteki + simgesine dokunun
3️⃣ Kanala Abone Ol butonuna basın veya kanal adını girin

📡 Kanal: yarimada-sim-giris-ozdere

✅ Ücretsiz, kurulumu kolay, SMS gelmese bile bildirim alırsınız.
